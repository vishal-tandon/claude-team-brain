# Perspectives: Index

Perspectives are **reasoning lenses**, not personas. Loading a perspective
means Claude reasons through that role's concerns and priorities. It does not
adopt that role's interaction style or voice.

A perspective answers: "What would a [role] notice, prioritise, or push back
on here?"

See `_TEMPLATE.md` for the format. See `example-perspective.md` for a worked
example.
